# TMF738 Revenue Sharing Model Management

TM Forum Open API (Apache 2.0) Revenue Sharing Model Management API.

Revenue Sharing Model Management API goal is to provide the ability to manage a revenue sharing model. This revenue sharing model allows to contains revenue sharing models that define how revenue is shared by an enterprise with one or more other parties.

The resources in this repository are a mirror of the official versions, published here for your convenience.
The authoritative source of information for this API is: https://www.tmforum.org/oda/open-apis/directory/
